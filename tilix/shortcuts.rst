Shortcuts
=========

Session
-------

* ``CTRL + ALT + 1`` - Switch to first session
* ``CTRL + ALT + 2`` - Switch to second session
* ``CTRL + ALT + 3`` - Switch to third session
* ``CTRL + ALT + 4`` - Switch to fourth session
* ``CTRL + ALT + 5`` - Switch to fifth session
* ``CTRL + ALT + 6`` - Switch to sixth session
* ``CTRL + ALT + 7`` - Switch to seventh session
* ``CTRL + ALT + 8`` - Switch to eighth session
* ``CTRL + ALT + 9`` - Switch to ninth session
* ``CTRL + ALT + 0`` - Switch to tenth session
* ``CTRL + PAGE DOWN`` - Switch to next session
* ``CTRL + PAGE UP`` - Switch to previous session
* ``CTRL + SHIFT + O`` - Open saved session
* ``CTRL + SHIFT + Q`` - Close current session
* ``CTRL + SHIFT + S`` - Save current session (explicitly as JSON file)
* ``CTRL + SHIFT + T`` - Open new session
* ``CTRL + SHIFT + PAGE DOWN`` - Move current session forwards
* ``CTRL + SHIFT + PAGE UP`` - Move current session backwards
* ``F12`` - Toggle session sidebar
