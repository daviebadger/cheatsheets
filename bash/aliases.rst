Aliases
=======

* ``alias`` - Show all aliases created via ``alias`` command
* ``alias alias_name='<command>'`` - Create one or more aliases
* ``unalias <alias_name>`` - Unalias one or more aliases
* ``unalias -a`` - Unalias all aliases
